# 🤖 AI Trading Terminal

A professional AI-powered stock analysis dashboard.

## 🚀 Features
- Candlestick charts
- RSI & MACD indicators
- AI Buy/Sell/Hold decision
- Confidence scoring
- Clean dark UI

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py  

## 🌐 Deploy
Push to GitHub and deploy on Streamlit Cloud.
